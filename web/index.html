EfficientSAM Segmentation

이미지 업로드 후 포인트/박스 프롬프트로 세그멘테이션을 실행하는 단일 데모 페이지입니다.

사용 방법

  1. 아래 데모 화면에서 이미지 업로드
  2. Point 또는 Box 모드 선택
  3. 이미지 클릭으로 프롬프트 입력 후 Run Segmentation
